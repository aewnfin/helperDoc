移动 C 盘下的 Users 文件夹

  • 概述

概述

 在用户处于登录状态下时， “Users 文件夹” 被系统所使用，而无法被正常移动

# 切换目录到c:\windows\system32；
cd c:\windows\system32

# 更改文件名为magnify.exe的所有者为当前用户；
takeown /f osk.exe

# 将当前用户所在的组（管理员组）赋予完全访问权限；
icacls osk.exe /grant administrators:F

# 将可执行文件magnify.exe更名为magnify_back.exe；
ren osk.exe osk1.exe

# 将cmd.exe复制并更名为magnify.exe；
mklink osk.exe cmd.exe
